

## Preview

[View Demo PDF](streamlit.pdf)

This project implements the SIR (Susceptible-Infected-Recovered) model to simulate the spread of infectious diseases through a population.

Users can adjust key parameters such as:
- Population size
- Infection rate (β)
- Recovery rate (γ)
- Simulation duration

The simulator generates real-time visualizations showing how disease outbreaks evolve over time.

## Features

- Interactive sliders
- Real-time outbreak visualization
- SIR epidemiological model
- Adjustable disease parameters
- Population dynamics graphs

## Technologies Used

- Python
- Streamlit
- Matplotlib

## How to Run

1. Clone the repository

```bash
git clone https://github.com/aayuuuushii/DiseaseSpreadSimulator.git
```

2. Install dependencies

```bash
pip install streamlit matplotlib
```

3. Run the application

```bash
streamlit run app.py
```
